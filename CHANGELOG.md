# Changelog

All notable changes to this project will be documented in this file.

The format is intentionally simple.

## [0.1.0] - 2026-04-13

Initial public v1 release candidate.

- Rust CLI with `doctor`, `run`, `inspect`, and `clean`
- `tmux`-backed lane orchestration
- deterministic manifest and run archive structure
- fail-closed lane result parser
- basic synthesis output
- initial docs for architecture, install, and product positioning
